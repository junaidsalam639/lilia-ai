# Lilia AI - Brain Aneurysm Diagnosis System

## Technologies Used

- Frontend: React 18 + Vite
- AI Models: Convolutional Neural Networks (CNN) + Large Language Models (LLM)
- Medical Imaging: Cranial CT Angiogram analysis
